# Browser Evidence Template

Observed:
- one or two concrete browser facts

Likely cause:
- one bounded explanation tied to the evidence

Next check:
- one reproducible validation step

Evidence:
- artifact paths, test names, screenshots, warnings, or short repro notes

Good evidence:
- "Restart button works, but `R` key does nothing in Chromium."
- "Collision increments score twice in `game.spec.ts` when two overlap events fire in one frame."

Weak evidence:
- "The game feels buggy."
- "Probably a state issue somewhere."
