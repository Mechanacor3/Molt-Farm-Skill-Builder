# Game Polish Pass

Use this skill when:
- The game is already playable and reasonably stable.
- The next gain comes from readability, feedback, animation, sound, or visual clarity.
- The user wants the game to feel more intentional without changing core scope.

Instructions:
1. Polish last. If the main loop is still unreliable, switch to a different skill first.
2. Focus on one or two polish goals per pass:
   - clearer feedback on hit, win, or loss
   - more legible UI or score state
   - tighter animation timing
   - better restart or onboarding cues
3. Keep polish in service of comprehension or feel. Avoid ornamental work that hides unresolved gameplay problems.
4. Report with these fields:
   polish_goal:
   change:
   user_visible_effect:
   recheck:
5. When useful, read `@./references/polish-heuristics.md`.
